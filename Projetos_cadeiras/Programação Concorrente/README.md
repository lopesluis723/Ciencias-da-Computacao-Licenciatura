# PC
## Jogo Multijogador (Agar.io-like)

Projeto de um jogo multijogador em tempo real desenvolvido em Erlang (servidor) e Java/Processing (cliente).

Os jogadores controlam um avatar num mapa 2D, podendo ganhar massa ao apanhar objetos e outros jogadores.

O servidor trata da lógica do jogo, matchmaking e pontuação, enquanto o cliente fornece a interface gráfica.

Comunicação feita através de sockets TCP.

## Tecnologias
- Erlang
- Java / Processing
- Sockets TCP
