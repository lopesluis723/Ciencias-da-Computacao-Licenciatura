# Trabalho-de-POO- SpotifUM
Projeto desenvolvido em Java para simular uma plataforma de streaming de música inspirada no Spotify. Inclui gestão de utilizadores, músicas, álbuns e playlists, bem como funcionalidades de reprodução, estatísticas e persistência de dados.

Tecnologias: Java, Programação Orientada aos Objetos e Serialização.
